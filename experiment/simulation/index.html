Virtual Lab Guide

Getting Started

  1. Select Device: Choose between ESP32 or ESP8266
  2. Configure LoRaWAN: Generate or enter DevEUI, AppEUI, and AppKey
  3. Set Parameters: Adjust spreading factor, bandwidth, and coding rate
  4. Enable Sensors: Toggle which sensors to simulate
  5. Activate Device: Click "Activate Device" to join the network
  6. Start Transmission: Click the play button to begin sending data

Understanding LoRaWAN

Spreading Factor (SF): Higher values increase range but reduce data rate. SF7 is fastest, SF12 has longest range.

Bandwidth: Wider bandwidth = higher data rate but less sensitivity.

Coding Rate: Forward error correction. Higher values add redundancy for reliability.

Network Components

  • End Device: Your ESP32/ESP8266 with sensors
  • Gateway: Receives LoRa packets and forwards to network server
  • Network Server: Manages devices, security, and routing
  • Application Server: Processes and stores sensor data
