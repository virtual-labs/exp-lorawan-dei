### About the Experiment 

This experiment focuses on developing an IoT application using LoRaWAN communication technology with ESP8266/ESP32 modules.  
It demonstrates long-range, low-power wireless communication suitable for large-scale IoT deployments.  
Students learn the architecture of LoRaWAN including end devices, gateways, and network servers.  
The experiment introduces concepts of low-power wide-area networks (LPWAN) and secure data transmission.  
The simulation provides practical understanding of energy-efficient and long-distance IoT communication systems.
